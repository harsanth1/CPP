This repository contains a collection of simple C++ programs written for practice and learning.
Each file focuses on a specific concept such as basics, arrays, functions, and common algorithms.The purpose of this repo is to improve C++ coding skills, understand core concepts, and build a strong foundation through small practice programs.

Topics Covered:
Basic C++ programs
Loops and conditional statements
Functions & recursion
Arrays and strings
Pointers & memory concepts
Small algorithms and pattern printing
